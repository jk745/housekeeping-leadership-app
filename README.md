# 房務部轉任領導系統｜Codex 專案包

這個專案包是把「從客務部櫃檯主任轉任房務部」的所有觀念、做法、讀書方法、Notion 架構與進度表，整理成 Codex 桌面端可以讀取、維護、擴充的資料夾。

## 這個專案包適合做什麼？

你可以用 Codex 協助你：

- 把 Owen《如何領導》的讀書筆記轉成房務部上任行動。
- 整理 30 天轉任準備進度。
- 記錄房務現場觀察。
- 建立房務團隊信任。
- 追蹤客務與房務銜接的改善議題。
- 產出主管回報稿。
- 整理成一頁式房務部領導手冊。
- 幫你把內容轉成 Notion 資料庫欄位、CSV 或模板。

## 使用方式

1. 將整個資料夾解壓縮到你的電腦。
2. 用 Codex 桌面端開啟此資料夾。
3. 先讓 Codex 讀取 `AGENTS.md`。
4. 打開 `prompts/01_Codex啟動提示詞.md`，把內容貼到 Codex 對話中。
5. 之後你可以把現場觀察、讀書筆記或主管交辦事項貼進 Codex，請它依照本專案規則整理。
