# 核心觀點與 IPA 領導框架

## 先記住這句話

> 你不是從零開始。你缺的不是領導能力，而是房務現場語言與實作細節的重新熟悉。

## 你的真正問題

你的問題不是：

> 我會不會鋪床、清房、查房都很熟？

你真正面對的是：

> 我如何在不假裝自己最懂房務實作的情況下，仍然成為一個讓房務團隊願意信任、願意跟隨的主管？

## 轉任定位

你不是要馬上成為最會做房的人，而是要成為：

- 最能理解現場的人
- 最能整合團隊的人
- 最能穩定品質的人
- 最能把客務視角轉化為房務管理價值的人

## 你的既有優勢

| 你在客務累積的能力 | 對房務領導的價值 |
|---|---|
| 了解客人感受 | 能判斷房務品質如何影響住宿體驗 |
| 熟悉客訴痛點 | 能抓出房況、清潔、備品、維修的風險 |
| 理解入住尖峰壓力 | 能設計更順的房況溝通方式 |
| 熟悉跨部門協作 | 能改善客務與房務的銜接 |
| 擁有櫃檯主任經驗 | 已具備基礎管理、溝通與危機處理能力 |

## IPA 框架

### I｜構想 Idea

你要問自己：

- 我想把房務部帶成什麼樣子？
- 團隊穩定代表什麼？
- 客人入住時，房務品質最容易被感受到的地方是什麼？
- 我希望客務與房務之間的合作變成什麼樣子？

建議方向：

> 房況穩定、細節可靠、同仁有安全感、與客務合作順暢。

### P｜人才 People

你要先贏得信任，不要急著證明權威。

可以對團隊說：

> 我過去主要在客務櫃檯，房務實作我需要重新熟悉，所以前期我會多請教大家。但我也會把客務端看到的客人需求、房況銜接和跨部門溝通經驗帶進來，讓我們一起把流程做得更順。

### A｜行動 Action

每個觀念都要轉成下一步。

| 觀念 | 下一步行動 |
|---|---|
| 要理解現場 | 跟房一次、看查房一次 |
| 要建立信任 | 和房務主管、領班、資深同仁各談一次 |
| 要改善跨部門合作 | 記錄客務與房務最常卡住的 3～5 個問題 |
| 要穩定品質 | 先看流程，不急著改流程 |

## 五個領導原則

1. 不假裝最懂現場，但要最快補上理解。
2. 先尊重原有經驗，再談改善。
3. 用客人視角管理房務品質。
4. 重視同仁辛苦、速度、細節與情緒壓力。
5. 用清楚流程、穩定溝通、具體行動建立信任。

## 最終目標

> 我的目標，不是成為最會做房的人，而是成為最能理解現場、整合團隊、穩定品質的人。
