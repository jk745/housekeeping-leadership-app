# 團隊信任建立策略

你從客務轉任房務，前期最重要的不是證明自己很懂，而是讓團隊感覺你願意理解、願意學、也願意負責。

## 你的開場定位

可以這樣說：

> 我過去主要在客務櫃檯，房務實作我需要重新熟悉，所以前期我會多請教大家。但我也會把客務端看到的客人需求、房況銜接和跨部門溝通經驗帶進來，讓我們一起把流程做得更順。

## 不要說的話

避免：

- 「我以前在櫃檯都覺得你們房務很慢。」
- 「這個流程應該很簡單吧？」
- 「我來之後要全部重新整理。」
- 「為什麼以前都這樣做？」

## 比較好的問法

改成：

- 「這個流程以前這樣設計，主要是為了解決什麼問題？」
- 「你們覺得最卡的是哪一段？」
- 「如果從現場角度看，客務端哪裡可以配合得更好？」
- 「這件事如果要改善，哪個做法最不會增加大家負擔？」
- 「我想先理解你們的做法，再一起討論怎麼更順。」

## 信任建立地圖

| 對象 | 他們可能在意 | 你可以怎麼做 |
|---|---|---|
| 房務主管 | 你能不能穩住團隊 | 定期回報學習進度與觀察 |
| 房務領班 | 權責會不會被打亂 | 尊重帶班節奏，先確認分工 |
| 資深房務員 | 新主管不懂現場 | 跟房、請教、記錄，不急著批評 |
| 新進房務員 | 主管是否願意支持 | 了解訓練需求與困難 |
| 客務主管 | 房況溝通能否更順 | 建立固定溝通節點 |
| 工程同仁 | 維修通報是否清楚 | 釐清通報格式與急迫程度 |

## 三個信任行動

### 行動 1：先請教

> 「這段我想重新熟悉，可以請你帶我看一次現場實際怎麼做嗎？」

### 行動 2：先肯定

> 「我以前在客務端只看到結果，現在才更理解你們現場要處理的細節很多。」

### 行動 3：先回饋

> 「我整理了一下今天看到的幾個卡點，不是要馬上改，而是想先確認我理解得對不對。」

## 信任原則

> 先讓團隊覺得你看見他們的辛苦，再讓團隊相信你的管理是為了讓事情更順。
